    What is 3D printing?

    3D printing is a form of additive manufacturing which can make quick plastic protoypes in a relatively quick time compared to other ways of manufacturing

    How does 3D printing work?

    3D printing works by extruding layer by layer of material, increasing the Z-axis each time till the entire piece is done

    The Prepertaion Process

    First the model is exported as an STL from Fusion 360.

    Next the model is uploaded to Ultimaker Cura, a 3D printing slicer software

    The settings of the 3D print can be select at the sidebar. These are the seetings used for this Specific Print

    Thicker wall thickness and higher infill percentage can be used for prints that need higher strenght

    The printing Process

    The Gcode file is saved onto an SD card and inserted into the 3D printer SD card slot. After the Print File is selected on the 3D printer UI. After a period of about 5 Minutes where the heated bed is heating up it will home all axiss and start the print

    The Print is removed from the Bed then the supporrt material is removed with snippers
