    What is Laser cutting?

    Laser cutters are CNC machines that use a high intenseity laser to cut materials like ply wood and acrylic

    Music Box

    The parts of the music box are designed and put together in in Fusion 360

    The Profile of each side is exported as an .DXF file and arranged in librecad

    The .DXF file is saved in a thumbdrive ad transfered to the laser cutter. On the software we select the lines to be cut as hairlines, After that the settings can be set

    Settings used

    The Laser is set then the cutting can be run

    if the parts are not cutt through the process can be ran again, if not the parts can be removed and glued together
